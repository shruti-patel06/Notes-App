# Mini Notes App

A simple, elegant web-based note-taking application with pinning, tagging, and search features. Built with vanilla JavaScript, HTML, and CSS.

## Features

- **Add Notes**: Create notes with text and optional tags.
- **Edit Notes**: Click "Edit" to modify note content inline.
- **Delete Notes**: Remove notes with the delete button.
- **Pin Notes**: Pin important notes to keep them at the top.
- **Tags**: Add comma-separated tags to notes for organization.
- **Search**: Search notes by text content or tags.
- **Timestamps**: Automatic date/time stamps for each note.
- **Persistent Storage**: Notes are saved in localStorage.
- **Responsive Design**: Works on desktop and mobile devices.

## Technologies Used

- **HTML5**: Structure and markup.
- **CSS3**: Styling with modern gradients, shadows, and responsive design.
- **JavaScript (ES6+)**: Logic for note management, DOM manipulation, and localStorage.

## Installation

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. No server required—runs entirely in the browser.

## Usage

1. **Adding a Note**:
   - Type your note in the "Type your thoughts..." field.
   - Optionally, add tags in the "Tags (comma separated)" field (e.g., "work, urgent").
   - Click "Add" to create the note.

2. **Editing a Note**:
   - Click the "Edit" button on a note.
   - Modify the text in the input field.
   - Press Enter or click "Save" to confirm, or Escape to cancel.

3. **Pinning a Note**:
   - Click "Pin" to move the note to the top with a highlight.
   - Click "Unpin" to return it to its original position.

4. **Searching Notes**:
   - Use the search bar to filter notes by text or tags.

5. **Deleting a Note**:
   - Click "Delete" to remove the note.
